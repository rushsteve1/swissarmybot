Help

You can use the both with the slash commands /quote and /bigmoji. The commands should guide you from there and prevent you from messing things up.

Referencing quotes (for getting and removing them) is done with their ID number. ID numbers are unique across all of SAB, not just a single person like they used to be.

BigMoji can be used inline just like regular emoji like this :emoji-name:. BigMoji also can't have spaces in their names, they'll just be removed if they do.

The /random endpoint can be used to post a random quote. This is primarily used for the Quote of the Day. Ask Steve if you need to know how it works for some reason.
